# Panda Racing 3D

**This repository contains ONLY the Panda 3D Racing game website.**

## Quick Start

```bash
# Clone this repository
git clone https://github.com/pandaaa2705/pandagaming.git

# Navigate to project folder
cd panda-racing-3d

# Start local server
python -m http.server 8000

# Open in browser
http://localhost:8000
```

## Project Structure

- `index.html` - Main game page
- `src/` - JavaScript game logic
- `styles/` - CSS styling
- `assets/` - Game assets (models, sounds)

## This Repository Contains ONLY

- Panda 3D Racing game
- No other websites or projects

## How to Add Your Models

1. Go to the `assets/models` folder.
2. Paste your Panda model and rename it to `panda.glb`.
3. Paste your Car model and rename it to `car.glb`.
