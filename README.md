# Panda Gaming

**This repository contains ONLY the Panda 3D Racing game.**

## Panda 3D Racing

A 3D racing game with Three.js featuring luxury cars and pandas.

### Quick Start

```bash
git clone https://github.com/pandaaa2705/pandagaming.git
cd panda-racing-3d
python -m http.server 8000
# Open http://localhost:8000
```

### Location

All game files are in the `panda-racing-3d/` folder:
- `index.html` - Main game page
- `src/` - JavaScript game logic
- `styles/` - CSS styling
- `assets/` - Game assets
